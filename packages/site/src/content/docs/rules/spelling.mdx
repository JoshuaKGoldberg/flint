## Presets

### `logical`

Rules that run CSpell to find likely spell-checking mistakes.

```ts title="flint.config.ts"
import { defineConfig, globs, spelling } from "flint";

export default defineConfig({
	use: [
		{
			files: globs.all,
			rules: spelling.presets.logical,
		},
	],
});
```
