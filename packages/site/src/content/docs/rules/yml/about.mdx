Flint's YML plugin supports linting `.yaml`/`.yml` source code.
It comes provided with the `flint` npm package.

## Presets

### `logical`

Rules that find bugs and enforce good YML practices for most-to-all YML files.

```ts title="flint.config.ts"
import { defineConfig, yml } from "flint";

export default defineConfig({
	use: [
		{
			files: yml.files.all,
			rules: yml.presets.logical,
		},
	],
});
```
